### 上传文件 api 说明文档

> 线上部署 https://btea.site/fileApi

**_上传文件_**
_接口地址_：`/upload`  
_请求类型_： `post`  
_参数_：`formdata`

**生成 svg 图片**  
_接口地址_：`/image`  
_请求类型_: `get`  
_必选参数_：  
`type`：生成 svg 图片类型

_调用栗子_: `/image?username=btea&type=text&text=天空高远，大风吟唱······`

**获取图片内容**  
_接口地址_：`/images/`  
_请求类型_：`get`  
_调用栗子_：`/images/c.jpg`

**获取文件内容**  
_接口地址_：`/files/`  
_请求类型_：`get`  
_调用栗子_: `/files/a.txt`
